This project is a console-based Social Media Application developed using Java.
It demonstrates Low Level Design (LLD) concepts and basic Object-Oriented Programming (OOP) principles.
Technologies Used

Java
Object-Oriented Programming (OOP)
Low Level Design (LLD)
Eclipse / VS Code (for development)
Git & GitHub (for version control)
